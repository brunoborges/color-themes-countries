## [1.0.0] - 2024-11-04

### Added
- Initial release with 20 country-inspired color themes
- Dark and Light variants for each country (40 themes total)
- Themes for: Argentina, Australia, Brazil, Canada, China, France, Germany, India, Israel, Japan, Netherlands, Poland, Russia, Singapore, South Korea, Spain, Sweden, Ukraine, United Kingdom, United States
- Modern, soft color palettes inspired by country flags
- Automated GitHub Actions workflows for build and release
- Comprehensive documentation (README, INSTALLATION, QUICKSTART, PUBLISHING, PROJECT_STRUCTURE)
